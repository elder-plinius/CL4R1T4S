# Changelog

All notable additions and changes to the CL4R1T4S prompt archive will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [Unreleased]

### Added
- `INDEX.md` — Full navigable index of all prompts by company
- `CONTRIBUTING.md` — Contribution guidelines with naming conventions and metadata template
- `CODE_OF_CONDUCT.md` — Community standards
- `CHANGELOG.md` — This file
- `.gitignore` — Ignore OS/editor artifacts
- `LICENSE-BURGESS` — Separate MIT license for the Burgess Principle directory
- GitHub Actions CI workflow for validating PR submissions

### Changed
- Standardized all filenames to use ISO 8601 dates (`YYYY-MM-DD`)
- Renamed `.mkd` files to `.md` for consistency
- Added `.md` extension to files that had no extension
- Renamed `BURGESS PRINCIPLE` → `BURGESS-PRINCIPLE` and `VERCEL V0` → `VERCEL-V0` (no spaces in directory names)
- Improved `README.md` structure with index link, licensing section, and prompt-injection test label
- Added metadata headers to all prompt files
